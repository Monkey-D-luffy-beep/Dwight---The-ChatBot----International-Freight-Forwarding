## Information Required for Quotes

### Basic Information
| Field | Details Needed |
|-------|----------------|
| Origin | City, country, pickup address |
| Destination | City, country, delivery address |
| Cargo Description | What you're shipping |
| HS Code | Harmonized System code (if known) |
| Weight | Gross weight in kg |
| Dimensions | Length × Width × Height per package |
| Packages | Number and type of packages |
| Incoterms | FOB, CIF, EXW, etc. |

### Additional Information (if applicable)
- Special handling requirements
- Temperature requirements (for cold chain)
- Hazardous goods details
- Insurance requirements
- Preferred shipping dates
- Urgency level

### For Accurate LCL Quotes
- Total volume in CBM (Cubic Meters)
- Total weight in kg
- Commodity type and value

### For Accurate FCL Quotes
- Container type required (20'/40'/40'HC/Reefer)
- Number of containers
- Commodity and weight

### For Air Freight Quotes
- Actual weight
- Package dimensions for volumetric calculation
- Urgency (standard/express)
- Any special requirements

## Types of Quotes

### Spot Quote
- One-time shipment pricing
- Valid for limited period
- Market-based rates

### Contract Quote
- Volume-committed pricing
- Extended validity
- Preferential rates
- Suitable for regular shippers

### All-In Quote
- Single price covering origin to destination
- Includes all charges (freight, handling, customs)
- Simplified budgeting

### Break-Down Quote
- Individual cost components listed
- Transparency on each charge
- Useful for cost analysis

## Frequently Asked Questions

### Q: How quickly can I get a quote?
**A:** FreightJar provides instant quotes. Direct inquiries typically receive responses within 24 business hours.

### Q: Is there a minimum shipment size?
**A:** No minimum for LCL or air freight. FCL requires booking at least one container.

### Q: Can I get quotes for multiple destinations?
**A:** Yes, provide all destinations and we'll quote each route.

### Q: Are customs charges included in quotes?
**A:** We can provide all-in quotes including customs, or separate customs quotations as needed.

### Q: How do I compare quotes from different forwarders?
**A:** Ensure you're comparing same scope (origin charges, freight, destination charges, customs). Ask for itemized breakdowns.
