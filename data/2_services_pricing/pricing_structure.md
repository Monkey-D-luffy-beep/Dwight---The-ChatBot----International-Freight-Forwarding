## Sea Freight Pricing

### FCL (Full Container Load)

**Cost Components:**
| Component | Description |
|-----------|-------------|
| Ocean Freight | Base carrier charges |
| THC Origin | Terminal handling at loading port |
| THC Destination | Terminal handling at discharge port |
| BL Fee | Bill of Lading documentation |
| Seal Charges | Container security seal |
| CFS Charges | If using container freight station |
| Customs Fees | Clearance charges |
| Delivery | Final mile to destination |

**Factors Affecting FCL Rates:**
- Container type (20'/40'/40'HC/Reefer/Special)
- Trade lane and specific ports
- Carrier selection
- Seasonal demand
- Contract vs. spot rates
- Special cargo requirements

### LCL (Less than Container Load)

**Charging Basis:**
Rates typically calculated on **Revenue Ton (RT)**:
- 1 RT = 1 CBM (Cubic Meter) or 1,000 KG, whichever is higher

**Cost Components:**
| Component | Description |
|-----------|-------------|
| Freight Rate | Per CBM/RT charges |
| Origin CFS | Cargo handling at origin |
| Destination CFS | Cargo handling at destination |
| Documentation | Bill of Lading, etc. |
| Customs | Clearance fees |
| Delivery | Final delivery charges |

**Factors Affecting LCL Rates:**
- Volume and weight of cargo
- Origin and destination
- Commodity type
- Consolidation frequency

## Air Freight Pricing

### Charging Basis
Chargeable Weight = Higher of:
- **Actual Weight** (Gross weight in kg)
- **Volumetric Weight** (L × W × H in cm ÷ 6000)

### Cost Components
| Component | Description |
|-----------|-------------|
| Air Freight | Per kg charges |
| Fuel Surcharge | Variable, fuel-indexed |
| Security Surcharge | Mandatory screening |
| AWB Fee | Airway Bill charges |
| Origin Handling | Airport handling fees |
| Destination Handling | Arrival handling |
| Customs | Clearance charges |
| Delivery | Door delivery (if applicable) |

### Factors Affecting Air Rates
- Origin and destination airports
- Direct vs. connecting flights
- Airline selection
- Urgency level
- Special cargo (DG, perishables, etc.)
- Weight breaks (higher volumes = better rates)

### Weight Breaks (Typical)
- Minimum: Up to 45 kg
- Normal: 45-100 kg
- 100 kg break
- 300 kg break
- 500 kg break
- 1,000 kg break

*Higher volumes typically command better per-kg rates*

## Customs Clearance Pricing

### Service Fee Basis
- Per shipment flat fee
- Complexity-based pricing
- Volume/value-based for larger shipments

### Components
| Service | Basis |
|---------|-------|
| Basic Clearance | Per Bill of Entry/Shipping Bill |
| Documentation | Complexity-based |
| EXIM Consultancy | Project/retainer basis |
| Duty Calculation | Included in clearance |
| Amendment Handling | Per amendment |
| Special Licenses | Case-by-case |

### Government Fees (Pass-through)
- Customs duty (as per tariff)
- IGST on imports
- Social Welfare Surcharge
- Examination fees (if required)
